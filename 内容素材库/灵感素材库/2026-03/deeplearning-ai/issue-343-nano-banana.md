# DeepLearning.AI The Batch - Issue 343

**日期**: February 27, 2026

## 重点新闻

### 1. Nano Banana 2 提升性能/价格比

Google推出了其旗舰图像生成器的更便宜、更快速的后继产品，在大约一半的价格下提供更高的交互性。

**核心要点**:
- Google推出了Nano Banana 2（正式名称为Gemini 3.1 Flash Image）
- 速度提升约4倍，成本约为前代产品的一半
- 在Arena.ai文本到图像排行榜上领先（1280 Elo）
- 通过Gemini应用、Google Ads、Google Antigravity和Flow免费（有限制）提供
- API：输入每百万令牌0.50美元，输出每张图0.045-0.151美元

### 2. 美国国防部抛弃Anthropic，转向OpenAI

OpenAI与美国政府签署了一份合同，提供可安全处理机密信息的AI系统，取代了Anthropic的Claude。

## Andrew Ng的分享

发布了Context Hub (chub)，这是一个开放上下文管理系统，为编码Agent提供所需的上下文。使用方法：

```bash
npm install -g @aisuite/chub
chub search openai
chub get openai/chat --lang py
```



升级世界模型形态，打造具身智能所需的动作驱动的世界模型

——不只是重建一个世界

上图来自上海AI Lab等团队近期发表的成果SoMA。

世界模型（ World Model ）是一类能够刻画环境动态演化规律的生成模型，近年来在视频生成与具身智能等领域展现出重要价值。

SoMA 以 3D 高斯泼溅 (Gaussian Splatt ing) 作为物体表征方式，以层级图神经网络为仿真主干，构建了一个 轻量级的 动作驱动的世界模型 (action-conditioned world model) 。

在给定机械臂动作序列的条件下，它能够进行物理仿真级别的世界状态演化预测，并同时输出高质量、多视角的一致性渲染结果。

## 简介

动作驱动的世界模型 (action-conditioned world model）的核心目标，是学习环境的状态转移规律，使系统在给定动作条件下预测未来世界演化。

在视频生成与具身智能领域，这类模型正逐渐成为连接 “ 感知 — 控制 — 生成 ” 的关键桥梁。

然而，具身系统的训练高度依赖大规模交互数据，真实采集成本高昂，而纯仿真数据又往往存在物理与视觉不一致的问题。

SoMA 正是在这一背景下提出的。

它构建了一个 real-to-sim 神经仿真器，将真实观测转化为可控的世界模型，从而低成本生成高质量操作数据。

不同于仅做 4D 重建的方法（缺乏动作条件与可控性），也不同于依赖精确物理参数的传统可微仿真器（如 MPM 、 SPH 等）——

SoMA 将机器人、柔性物体与环境统一建模于同一仿真空间中。

具体而言， SoMA 以 3D 高斯泼溅作为物体表示，以层级图神经网络建模力驱动的动态传播机制，构建了一个 轻量级动作驱动的世界模型 (action-conditioned world model) 。

在给定机械臂动作序列后，模型能够在三维空间中推进物理一致的状态演化，并同步输出多视角、高一致性的渲染结果。

不同于传统仿真器需要显式估计材料参数、再将高斯泼溅转换为适配物理引擎的中间格式
，SoMA 直接在统一的三维表示上进行动力学建模与可微渲染。神经网络主干既减少了真实世界与仿真空间之间的对齐成本，也显著提升了视觉呈现质量。

它所构建的不只是一个“能跑起来”的仿真环境，而是一个更加贴近真实观测、同时又可被动作驱动的世界模型。
这为具身柔体操作任务提供了一个高质量、可扩展的数据生成环境，也为构建可控世界模型迈出了关键一步。

挑战

要构建一个既可控、可视、又物理一致的 real-to-sim 世界模型，尤其是在机器人柔性操作场景下，至少面临三类核心挑战：

real-to-sim 对齐与表征问题

真实观测来自相机与机器人系统，而仿真运行在一个理想化的物理空间中。

尺度不一致、坐标系不同、物理参数未知，都会导致 “看起来对，但控制不起来”的问题。与此同时，柔性物体具有连续形变特性，传统网格或粒子表示往往难以同时兼顾几何表达能力与可微仿真的稳定性。

因此，一个有效的 world model 既要解决真实与仿真空间的映射问题，也需要一种既连续又适合动力学传播的物体表示方式。

渲染质量与长序列稳定性问题。

仿真本质上是几何与动力学的演化，但在具身学习与数据生成场景中，最终输出往往是视频或多视角图像结果。

仅有几何一致性远远不够，还需要视觉层面的高保真与跨视角一致。同时，机器人操作往往是长时间序列过程，微小误差会在滚动预测中逐步放大，导致形变失真或动态漂移。

因此， world model 不仅要“能动”，还要 “动得久、看得清”。

具身交互与因果建模问题。

机器人操作本质上是力驱动的接触过程，而非纯视觉状态预测。

机器人、物体与环境之间存在显式的相互作用关系，尤其在柔性场景中，局部接触会引发全局形变传播。如果模型只学习表面像素变化，而没有建模交互与力的传播机制，就难以达到仿真级别的物理一致性。

此外，实际场景中还存在遮挡问题，仅依赖可见区域监督会进一步削弱动力学推断能力。

这一系列挑战意味着： 构建一个面向具身操作的world model，不仅是生成问题，也不仅是重建问题—— 而是一个几何表示、动力学建模与真实空间对齐三者耦合的系统工程 。

方法

SoMA 构建了一个“具身级别”的可控世界（如图所示）。

这个“世界”由三部分组成：

柔性操作对象、机械臂，以及三个固定视角的相机。

- 操作对象 用 3D Gaussian Splats 表达——既是几何表示，也是可渲染单元；
- 机械臂 不再是模型里的一块“视觉物体”，而是以动作序列的形式直接驱动世界演化；
- 相机 则一边提供多视角观测监督，一边通过标定完成真实空间与仿真空间之间的对齐。

围绕前面提到的三类核心挑战，SoMA做了三层对应设计。

第一层：解决 real-to-sim 的对齐与表示问题。

真实世界和仿真世界天然不一致 ——尺度不同、坐标不同、控制信号也无法直接作用。

SoMA 构建了一个“机器人条件化”的 real-to-sim 映射，把机器人、物体和环境统一到同一个仿真空间中。

同时，用 3D Gaussian Splats 作为连续表征，并在每个 splat 上编码隐式物理属性，使物体既能高质量渲染，又能在连续空间中承载可传播的动力学状态。

它不是先重建一个“看起来像”的几何，而是直接建立一个“可以被控制”的世界。

第二层：解决渲染质量与长序列稳定性问题。

仿真本质是几何与动力学，但具身训练需要的是视频级别的数据。

SoMA 在 3D 隐空间中进行动力学传播，同时直接从真实的多视角 2D 图像进行监督，而不是依赖由图像估计得到的 3D 轨迹作为中间监督信号。

通过Gaussian Splats 的可微渲染机制，建立了从图像到三维状态的反向传播路径，多视角一致性保证了几何恢复的稳定性。

同时，它采用多分辨率时间训练策略——先学粗粒度的长期动态，再细化短期变化。

既降低了长序列训练成本，也避免了误差在滚动预测中不断放大。

它不仅“动得起来”，还“动得久、看得清”。

第三层：解决具身交互与动力学建模问题。 机器人操作本质是力驱动的接触过程，而不是像素变化。

SoMA 使用层级图神经网络建模机器人—物体—环境之间的交互结构，让局部接触能够向全局形变传播。

再配合物理一致性约束与遮挡感知监督，即便在部分可见的情况下，也能维持合理的动力学演化。

它学习的不是表面变化，而是交互因果。

通过这三层结构设计，SoMA 实际上构建了一个面向机器人操作的世界模型：

给定机械臂动作序列，系统在物理一致的仿真空间中推进世界状态，同时生成多视角、高保真的可视化结果。

它不只是重建一个世界，而是构建了一个可以被控制、可以被演化、可以被训练的世界。

实验

我们在多个真实机器人操作场景上验证了 SoMA，包括绳子，布料，玩偶以及更具挑战性的叠衣服任务（如图所示）。

测试同时覆盖 resimulation（训练轨迹重演） 与 generalization（未见动作泛化） 两种设置。

整体结果非常清晰： SoMA 在物理准确性与视觉质量上均显著优于现有方法。

在定量指标上， 研究从两方面评估模型表现：

- 视觉一致性（PSNR / SSIM / LPIPS）
- 几何准确性（AbsRel / RMSE 深度误差）

相比 PhysTwin（物理驱动方法）与 GausSim（神经模拟器），SoMA 在大多数场景中取得更高 PSNR、更低 LPIPS，同时深度误差显著下降。

这说明模型不仅“看起来更清晰”，而且三维结构更准确。

尤其在长序列任务（如叠衣服）中，SoMA 的误差累积更小，动态演化更加稳定，而对比方法在复杂形变下往往出现结构漂移或局部崩塌。

在视觉质量上，差异更加明显。

传统神经模拟方法通常在低维状态空间中预测节点位置，再通过独立渲染器生成图像；而 SoMA 是直接在 3D Gaussian Splats 表征上进行动力学更新。

- 仿真与渲染在同一空间内完成
- 每一步物理更新都会直接影响可视化结果 因此，多视角渲染天然保持一致性，细节更加锐利，形变更加连续。

从定性结果来看，SoMA 在布料褶皱、绳子弯曲、玩偶受力变形等细节上都呈现出更自然的动态变化； 而对比方法往往存在模糊、结构断裂或局部抖动 。

- 更准确的物理演化
- 更稳定的长序列模拟
- 更高保真的多视角渲染

这使它不仅是一个仿真器，更是一个可以直接产出高质量训练数据的可控世界生成系统。

结语

从更大的视角来看，SoMA 其实站在“仿真器”和“世界模型”这两类方法之间。

- 一类是传统物理仿真器，如可微分的MPM，弹簧质点模型等

仿真与渲染共享同一 3D Gaussian Splats 表征，并在其中引入隐式物理属性编码，使几何、外观与动力学状态天然耦合，从而同时提升仿真精度、渲染质量与多视角一致性。

它们拥有明确的物理建模能力，但往往依赖精确的物理参数设定，real-to-sim 对齐成本高，部署流程复杂。

一旦场景或物体变化，往往需要重新标定或调整参数。 同时，其渲染通常独立于仿真系统，视觉质量和多视角一致性并非设计核心。

相比之下，SoMA 直接从真实多视角观测出发，建立机器人条件化的 real-to-sim 映射，无需显式恢复精确材料参数即可完成物理一致的动态演化。

仿真与渲染共享同一 3D Gaussian Splats 表征，使几何与外观天然耦合，渲染质量与多视角一致性显著提升。

它不是“先做仿真，再做渲染”，而是在一个统一表示中同时完成两件事。

- 另一类方法是近年来兴起的 diffusion-based world models。

它们在视频生成上表现出强大的感知能力，但本质上仍停留在像素空间或隐变量空间的动态建模。

几何结构往往是隐式的，物理因果关系并未显式建模，多视角一致性也难以保证。 模型可以生成“看起来合理”的未来帧，却难以对动作施加稳定、物理一致的响应。

相比之下，SoMA 的世界演化发生在显式三维空间中：

几何是连续的，动力学是力驱动的，状态更新是可解释的；

动作输入直接影响物理状态，再通过可微渲染输出图像；

几何一致性、物理一致性与多视角一致性在同一框架下得到统一。

因此，SoMA 并不仅仅是一个柔性物体模拟器。

它更接近一种“动作驱动的世界模型”的形态： 动作作为因，三维物理状态作为中间变量，多视角可视化作为果 。

这种结构，使世界模型不再只是生成未来画面，而是成为一个可被控制、可被演化、可用于训练策略的基础设施。

如果说早期的 world model 回答的是—— “世界将如何变化？”

那么 SoMA 所代表的方向回答的是—— “当我施加一个动作，世界会如何真实地变化？”

这或许正是具身智能真正需要的世界模型。

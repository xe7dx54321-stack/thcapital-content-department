来源 ｜ 新智元

就在昨天下午，OpenClaw已经超越了所有GitHub开源软件项目的星标数（Star），正式加冕史上最受欢迎开源项目！

更令人瞩目的是，OpenClaw的数据是以近乎垂直的直线飞速飙升的，爆发后仅仅用了今年前两个月，就超越了如Linux和React等众多持续更新了十多年的超大型基建项目！

注意力的通货膨胀

要理解这场风暴，我们需要重新审视「星标」在今天的真实意义。

它早已超越了单纯的下载量与日活指标范畴，也脱离了严谨技术背书的语境。

在开源世界里，点下一颗星代表着成本极低的表态，它介于收藏、点赞与已读之间。

揭开排行榜的真实面貌往往令人意外。

长期占据榜首的往往是各种资源汇总、教程和书单（例如拥有47万星标的build-your-own-x仓库），鲜少看到基础软件的影子。

当你把这些资源库和活动类项目剔除，真正审视那些可以被安装运行的基础软件时，OpenClaw的登顶便显得极具颠覆性。

React花了十三年时间，依靠无数求职要求、企业架构选型和培训班的反复锤炼，才攒下逾24万颗星，这代表了极深的基础设施渗透率。

OpenClaw达到同样的高度只用了短短一百天。

这背后昭示着注意力经济在开源 社区 的彻底爆发。

一只名叫Molty的龙虾，正在狂飙

了解这场风暴的中心，需要先看清OpenClaw的本质。

由奥地利独立开发者Peter Steinberger在2025年11月推出的OpenClaw，定位为完全开源且本地运行的AI Agent框架。

它彻底脱离了封闭的云端生态，直接将大语言模型接入人们每天使用的WhatsApp、Telegram、Discord甚至iMessage等通讯工具。

它具备真正的行动力。

普通用户的电脑在它眼中变作了可以肆意发挥的沙盒，执行终端命令、读写文件、收发邮件乃至代替主人管理日程，一切均在自然的对话间完成。

这个项目自带一种属于极客社区的荒诞与幽默。

它的吉祥物是一只名叫Molty的龙虾，社区里甚至流传着一句脱胎于科幻剧集的黑话「去角质（EXFOLIATE！）」。

这种亚文化让它迅速剥离了枯燥的软件工具标签，变成了一种数字图腾。

它经历了一场戏剧性的更名风波，起初名为Clawdbot的它遭到商标警告后改名Moltbot，最终才定型为今天大杀四方的OpenClaw。

每一次争议，客观上都化作了它病毒式传播的燃料。

破圈与失控的边缘

OpenClaw的引爆点不仅在于技术的突破，更在于受众的彻底下沉与场景的极度分散。

上一代明星项目（如Vue、Go、Kubernetes）的推广路径遵循着自上而下的逻辑，往往由技术主管拍板后团队跟进。

OpenClaw的蔓延却完全跨越了技术壁垒。

在社交网络上，你可以看到截然不同的两幅画面。

一位休产假的设计主管用单手在手机上通过它处理所有生活琐事，一位母亲让它在WhatsApp家庭群组里自动规划饮食和接送孩子。

与此同时，硬核开发者们正忙着把它改造成24小时无休的自动写代码机器。

伴随狂热而来的还有真实世界里的失控。

由于赋予了AI极高的系统权限，今年一月底，OpenClaw社区爆发了名为「ClawHavoc」的安全危机。

攻击者通过伪装的技能包，让大量暴露在公网的本地实例感染了恶意软件。

紧接着，部分科技巨头以异常使用导致服务降级为由，大规模限制了通过该工具接入云端大模型后端的开发者账号。

甚至有硅谷大厂在内部明令禁止员工在工作设备上运行该程序。

危险、漏洞以及外界的封锁非但没有扑灭OpenClaw的热度，反而坐实了它具备真正破坏性力量的地位。

在很多人眼中，这种游走在失控边缘的刺激感恰恰印证了人类正真真切切地活在未来。

时代的分水岭

如果把GitHub的历史排行榜拉长，你会看到两个截然不同的时代。

2013到2016年属于Web与云原生时代。那是工程师们为了构建更稳定的互联网大厦而添砖加瓦的黄金岁月。

2022年至今则进入了AI时代。

从AutoGPT到LangChain，再到今天的OpenClaw，收割星标的速度越发惊人。

走红的驱动力已经发生根本转变。

工程师下载React源于构建商业系统的刚需。

当普通人将目光投向OpenClaw时，驱动力已经变成了纯粹的好奇、极度的兴奋，乃至对未知的隐忧。

尾声

OpenClaw算得上GitHub上最重要的开源项目吗？

理性的答案依然是否定的。

拥有22万颗星的Linux至今默默支撑着全球绝大多数的服务器和智能设备，那是互联网跳动的稳定脉搏。

然而，在短短四个月内让超过24万人愿意为之驻足并点亮星标，OpenClaw已经证明了自己是当下技术情绪的最大公约数。

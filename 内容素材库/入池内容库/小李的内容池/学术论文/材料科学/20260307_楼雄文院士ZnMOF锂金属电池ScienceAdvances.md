## 一、研究背景与挑战

### 1. 无负极锂金属电池的价值

**无负极5 V级锂金属电池（LMBs）**有望实现高能量密度和安全性。

### 2. 核心挑战

在实际应用中，无负极锂金属电池面临严峻挑战：

| 挑战 | 说明 |
|------|------|
| 枝晶生长 | 锂沉积不均匀，形成枝晶 |
| 界面不稳定 | 固体电解质界面不稳定 |
| "死"锂积累 | 锂粉碎化导致容量损失 |
| 能量密度降低 | 使用铜集流体导致能量密度降低 |

---

## 二、核心研究成果

### 1. 创新方案

香港城市大学**楼雄文院士**等人展示了一种精心设计的**二维六方含锌金属有机框架（Zn-MOF）**，锚定在导电Ti₃C₂Tₓ MXene片上（MX/2D-Zn-MOF），作为无负极5 V级LMBs的**三维轻质主体**。

### 2. 关键机制

丰富的**富氮/富氟/富锌的亲锂位点**有效：
- 降低锂成核能垒
- 保证锂沉积均匀
- 稳定固体-电解质界面

相互连接的三维框架进一步：
- 空间上均匀化锂离子通量
- 降低局部电流密度
- 指导均匀的锂沉积
- 提供足够空间缓解结构变化
- 保证结构完整性，防止粉碎

## 三、材料设计与合成

### 1. MX/2D-Zn-MOF的形成过程

**轻量化柔性材料MX@2D-Zn-MOF**的合成过程：

1. 用HCl/LiF蚀刻剂对大块Ti₃AlC₂进行化学蚀刻
2. 进行温和的剥离，合成含有二维Ti₃C₂Tₓ片的MX胶体溶液
3. 将Zn²⁺离子加入到MX胶体溶液中
4. 通过静电相互作用形成Ti₃C₂Tₓ-Zn²⁺前驱体
5. 通过配位反应在MX片上原位生长Zn-MOF晶体

### 2. 三种Zn-MOF结构

通过调节反应介质中Zn²⁺和2-甲基咪唑的浓度，可以精确调节Zn-MOF的晶体结构和形貌：

| 类型 | 结构 | 形貌 |
|------|------|------|
| dodeca-Zn-MOF | 方钠石结构 | 菱形十二面体 |
| accordion-Zn-MOF风琴状 |
 | - | 手| **2D-Zn-MOF** | 金刚石结构 | 六方二维 |

## 四、DFT计算与机理研究

### 1. 亲锂性分析

计算得到的锂结合能（Eᵦ）：

| 材料 | 锂结合能 |
|------|----------|
| **2D-Zn-MOF** | **最高** |
| Ti₃C₂Fₓ | 其次 |
| Cu(111) | 传统基准 |

**2D-Zn-MOF的Eᵦ值最高**，具有优异的亲锂性，有利于均匀的锂成核和生长。

### 2. 成核过电位

| 电极 | 成核过电位 | 平台过电位 |
|------|-----------|-----------|
| MX/2D-Zn-MOF | 57.7 mV | 54.5 mV |
| MX/accordion-Zn-MOF | 65.6 mV | 62 mV |
| MX | 194.5 mV | 57.7 mV |

## 五、电化学性能

### 1. 循环稳定性

在不同电流密度和面容量下，MX/2D-Zn-MOF表现出更好的可逆性和循环稳定性：

| 条件 | 循环时间 |
|------|----------|
| 1.0 mA/cm², 1.0 mAh/cm² | 长期稳定 |
| 1.0 mA/cm², 5.0 mAh/cm² | **1800小时** |
| 0.5 mA/cm², 5.0 mAh/cm² | **2400小时** |

### 2. 对称电池性能

MX/2D-Zn-MOF-Li电极在不同电流密度下的电压滞后：

| 电流密度 | 电压滞后 |
|----------|----------|
| 1.0 mA/cm² | 29.4 mV |
| 2.0 mA/cm² | 49.9 mV |
| 3.0 mA/cm² | 70.9 mV |
| 5.0 mA/cm² | 112.3 mV |

## 六、无负极5V级锂金属电池性能

### 1. 全电池性能

组装无负极5 V级LMBs（LNMO正极 + MX/2D-Zn-MOF-Li负极）：

| 参数 | 数值 |
|------|------|
| 首次放电容量 | 125.8 mAh/g |
| 120次循环后容量保持率 | **98.3%** |
| 能量密度 | **538.6 Wh/kg** |

### 2. 对比结果

| 负极材料 | 120次循环后容量保持率 |
|----------|---------------------|
| **MX/2D-Zn-MOF-Li** | **98.3%** |
| MX/accordion-Zn-MOF-Li | 78.7% |
| MX-Li | 39.7% |

### 3. 软包电池应用

成功组装**0.37 Ah级MX/2D-Zn-MOF-Li||LNMO软包电池**：
- 能量密度：538.6 Wh/kg
- 循环性能稳定
- 成功为智能手机、LED灯带、风扇供电

## 七、行业意义

### 1. 技术突破

MX/2D-Zn-MOF作为三维轻质主体，有效解决了无负极锂金属电池的关键问题：
- 抑制枝晶生长
- 提高循环稳定性
- 增强能量密度

### 2. 实际应用价值

- 为高能量密度储能提供新方案
- 推动电动汽车、便携设备发展
- 加速新能源技术落地

## 行业大事件

### 1. 谷歌发布Nano Banana 2

北京时间27日凌晨，谷歌推出最新图像模型**Nano Banana 2**，使得高质量图像的生成：
- **更快**
- **更便宜**
- **更容易**

### 2. Nano Banana 2核心技术

官方宣称，Nano Banana 2（即Gemini 3.1 Flash图像模型）能够兼具：

| 能力 | 描述 |
|------|------|
| 图像生成质量 | Nano Banana Pro级别 |
| 推理能力 | 先进世界知识 |
| 生成速度 | Flash模型"闪电速度" |

### 3. 功能下放

一系列曾经专属于"Pro"系列模型的功能下放到最基础的版本：

| 功能 | 描述 |
|------|------|
| 角色一致性 | 最多5个角色的相貌一致 |
| 物体忠实 | 14个物体的忠实呈现 |
| 语义理解 | 自然语言理解能力提升 |

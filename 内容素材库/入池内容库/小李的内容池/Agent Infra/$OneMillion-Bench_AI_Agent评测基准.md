# $OneMillion-Bench：AI Agent 百万美元评测基准

> 整理日期：2026-03-12
> 标签：#Agent评测 #基准测试 #投资观点 #2026-03

---

## 核心内容

### 1. 评测基准的诞生
"如果有价值100万美金的顶级专家任务，AI能完成其中多少？答案是48万美金，而只需要100美元的Token费用。"

Humanlaya Data Lab联合北京通用人工智能研究院（BIGAI）、xbench、M-A-P，招募来自Morgan Stanley、世达（Skadden）、协和医院、中国电网、清华大学等顶级机构或学府的100+位资深专家，耗时2000+小时，构建了等价于人类专家工作价值百万美元级评测基准——$OneMillion-Bench。

### 2. 核心设计理念
"$OneMillion-Bench的核心想法很直白：用'人类专家的时间与成本'给任务定价，再用'是否满足专家要求'衡量模型交付质量——如果把AI当成'数字白领专家'，总价值100万美金的任务，模型到底能赚多少钱？"

该基准包含400道高难题目（200个英文题+200个中文题），覆盖金融、法律、医疗、自然科学与工业五大领域的92个三级领域。每道题都是真实行业场景下的开放专家任务。

### 3. 经济价值计算方式
"任务经济价值=资深专家完成该任务的耗时×专家时薪。"时薪锚定官方或行业权威数据，如中国部分城市人社局、美国劳动统计局，任务的耗时来自多个领域专家的共同评估。

### 4. 关键设计要点

#### 多样化真实场景 + 高价值任务
"我们在评测中引入用货币度量的'经济价值'，核心是衡量模型在真实世界中能创造多少可交付的经济价值。开放式问题很难用单一标准答案衡量，我们邀请一线资深专家将真实工作流拆解为细颗粒度考点：每道题设计15–35个考点，累计7000+考点。"

#### 非对称负分机制
"在考点分值设置上，我们采取+10 ~ -20的非对称考点分值：正向能力给分更克制，明确或致命错误惩罚更重。这套结构的效果更接近真实使用体感——做对不一定加分很多，但做错往往会带来更大代价。"

### 5. 当前模型表现
"目前最强模型通过率超过40%，在100万美元的任务上，大约可以产出50万美元，而完成任务的API成本也就100美元左右！"

"如果只看平均分，头部模型已经进入了合格区间（60%+），第二梯队也普遍在50%以上。但从通过率的视角来看，即使是排名第一的Claude Opus 4.6 Web Search，也骤降到43.5%，即只有不到45%的任务可以通过验收。"

### 6. 关键Insight

#### Insight 1：成绩已经合格，但离"可托付"还很远
"平均分像'考试成绩'，而通过率才是'上岗证'。距离可交付的专业任务仍有相当一段路程。"

#### Insight 2：Web Search是把双刃剑
"Web Search工具调用通常能显著补齐事实，但也会引入噪声与'看似权威的错误来源'。下一阶段竞争不只是'有没有搜索'，而是'会不会搜索'以及搜索工具的效果如何。"

#### Insight 3：复杂推理仍是通用瓶颈
"模型擅长写一段看起来连贯的解释，但一旦任务需要深层理解、多步演绎、或在巨大可能空间里探索，就仍会出现深度不足与准确性波动。"

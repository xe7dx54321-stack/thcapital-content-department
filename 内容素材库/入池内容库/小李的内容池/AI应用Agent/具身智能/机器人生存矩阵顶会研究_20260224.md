# 四项研究构建机器人"生存矩阵"：从IROS到AAAI

> 整理日期：2026-03-11
> 标签：#具身智能 #机器人 #IROS #ICRA #ICLR #AAAI #2026-02

---

## 核心内容

### 1. 核心挑战
"如何在动态、不确定甚至部分受损的环境中，持续、安全且灵巧地行动——已经成为具身智能又一轮的核心挑战。"

### 2. 四项顶会研究
**IROS 2025: Thinker - 视觉语言基础模型**
- 发文团队：优必选
- 亮点："定制数据与联合输入，突破 VLMs 机器人应用瓶颈获 SOTA"
- "针对大视觉语言模型（VLMs）应用于机器人领域时存在的视角混淆、视频结尾信息忽略等问题，研究者提出面向具身智能的视觉语言基础模型Thinker。"

**ICLR 2026: TMoW - 测试时混合世界模型**
- 发文团队：成均馆大学
- 亮点："动态环境测试时自适应，定义『具身智能 + 测试时适配』的新研究范式"
- "针对具身智能体在动态环境中适配性有限的问题，该研究扩展混合专家范式，提出TMoW框架。"

**ICRA 2026: DEFT - 故障主动轨迹扩散生成**
- 发文团队：科罗拉多大学
- 亮点："故障主动轨迹扩散生成"
- "针对机器人执行器故障易导致停机、现有方法难以适配任意故障的问题，该研究提出DEFT扩散策略框架，实现机器人的故障主动操作。"

**AAAI 2026: SDP - 技能条件扩散策略**
- 发文团队：香港大学
- 亮点："控技能驱动型机器人操作学习，首次将『技能』作为具身智能的核心中间层"
- "针对现有扩散策略依赖全局指令导致动作生成失准的问题，提出技能条件扩散策略（SDP）。"

### 3. 共性
- "所有研究均围绕『让智能体/机器人更灵活、更抗造』展开"
- "核心提升具身智能的场景容错率"

### 4. 分化方向
"从 全局感知理，到 环境动态预测，再到 内部故障容错 与 技能组合创新 的立体技术矩阵"

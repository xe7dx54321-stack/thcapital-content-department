# 丰田研究院VLA训练教程

**来源**: 机器之心
**日期**: 2026-03-07
**标签**: #VLA #机器人 #丰田研究院 #具身智能

## 核心内容

### 研究规模

- 4000小时机器人与人类操作数据
- 5000万个视觉-语言样本
- 89个不同策略模型
- 58000次仿真评估
- 2835次真机测试

### Co-training（协同训练）

**核心理念**：让机器人"博采众长"，从异构数据学习

### 五种数据模态

1. 标准视觉-语言数据
2. 机器人轨迹的语言标注
3. 跨具身机器人数据
4. 人类视频
5. 离散机器人动作Token

### 三种训练策略

1. 单阶段协同训练
2. 两阶段-仅首阶段协同
3. 两阶段-全协同

### 关键结论

**有效模态**：
- 标准视觉-语言数据
- 跨具身机器人数据
- 人类视频的语言标注

**无效**：
- 离散动作Token
- FAST Token

### 核心发现

- Co-training显著提升泛化性
- 组合模态训练使真机任务完成率提升45.3%
- 仿真环境未见任务成功率提升36.4%

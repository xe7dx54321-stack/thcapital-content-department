# Cursor版OpenClaw与小米版OpenClaw登场

## 核心内容

### 一、Cursor Automations

**功能发布：**
- Cursor推出类OpenClaw功能——Cursor Automations
- 让AI 7×24小时自动帮开发者审核、监控、修复代码
- 云端Agent驱动，使用自己的算力构建、测试和演示工作成果

**价格：**
- 云端Agent和编辑器里用的一样的token
- 享受开发者的Ultra订阅权益

**应用场景：**
1. 代码评审与监控
2. 日常事务处理

**内置Agent：**
1. 安全审查（Security review）- 每次代码推送时触发
2. 智能代码责任人（Agentic codeowners）- PR风险分级
3. 事件响应（Incident response）- PagerDuty告警触发
4. 变更每周汇总（Weekly summary of changes）
5. 测试覆盖（Test coverage）
6. 错误报告（Bug report triage）

**开发者评价：**
- "未来我们只需当个超厉害的机器人管理员就行"
- "Cursor快变成OpenClaw🦞了"

**安全风险：**
- 当Agent自主合并补丁却搞挂生产环境时，该由谁来担责？

### 二、Xiaomi miclaw

**发布：**
- 小米启动类OpenClaw的移动端系统级智能体
- 基于小米MiMo大模型构建
- 移动端AI交互测试产品
- 小范围封闭测试，仅支持小米17系列

**核心能力（四层架构）：**
1. 系统底层能力
2. 个人上下文理解
3. 生态互联
4. 自进化机制

**使用场景：**
1. 家庭设备管理 - "看看家里缺啥耗材"
2. 健康管理 - "帮我减脂3kg"
3. 起床提醒 - "起不来你就折腾"
4. 朋友回家自动准备环境

**团队背景：**
- MiMo大模型负责人福莉（原DeepSeek核心成员：罗、"天才少女"）
- 2025年11月加入小米出任MiMo大模型负责人
- MiMo-V2-Flash性能已可对标DeepSeek-V3.2

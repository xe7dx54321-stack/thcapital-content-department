# 微软Project Silica：玻璃中的永恒存储

> 整理日期：2026-03-10
> 标签：#微软 #ProjectSilica #玻璃存储 #数据存储 #2026-03

---

## 一、项目启动与早期探索（2017-2018）

### 1. 2017年：首次公开亮相

Project Silica在Microsoft Ignite 2017主旨演讲中首次被提及，作为一种未来的云存储技术，介绍了使用飞秒激光在石英玻璃中存储数据的概念。

这标志着微软开始探索利用新型介质进行长期数据存储的可能性。

### 2. 2018年：技术视频发布

2018年8月1日：发布早期技术视频，详细介绍了飞秒激光写入的核心概念。

同年，在USENIX HotStorage会议上发表论文，探讨玻璃作为新一代存储介质的可能性。

---

## 二、关键里程碑：首次概念验证（2019）

2019年2月/9月：微软研究院开始大规模招募系统研究人员，加速开发云时代的创新光学存储方案。

2019年11月：成功存储并读取电影《超人》(Superman)。这是该项目的一个重大里程碑。微软与华纳兄弟合作，将1978年版的电影《超人》（约75.6GB）完整存储在了一块75x75x2毫米的石英玻璃上并成功读取。

这证明了该技术存储大型商业资产的端到端可行性。

---

## 三、系统深化与文化遗产保护（2022-2024）

2022年6月：微软宣布与Elire Group和Global Music Vault合作，将重要的音乐数据（包括经典歌剧和现代音乐）存储在玻璃中，并计划将其埋在挪威斯瓦尔巴群岛的冰层下保存万年。

2023年10月：在SOSP 2023会议上发表论文，详细描述了Silica系统如何整合机器学习(ML)解码、新型机器人库设计以及针对Azure云工作负载的优化方案。

2024年5月：发布关于RASCAL机器人的研究，展示了专门为玻璃存储设计的可扩展、高冗余机器人系统。

2024年9月：启动"Golden Record 2.0"项目，与学生合作将人类多样性的数字档案编码到石英玻璃盘上，作为向旅行者号金唱片的致敬。

---

## 四、关键里程碑：系统完善与材料突破（2025-2026）

### 1. 2025年综合论文

2025年2月：在《ACM存储交易》(ACM Transactions on Storage)上发表综合性论文，系统性地描述了从介质到云API的完整硬件和软件共设计方案。

### 2. 2026年Nature论文

2026年2月18日：发表《Nature》论文并宣布研究阶段完成。这是该项目的最新重大里程碑。

---

## 五、2026年突破性进展

### 1. 材料突破

成功将技术从昂贵的合成石英玻璃扩展到廉价的硼硅酸盐玻璃（常用于厨房厨具）。这大大降低了成本，为商业化铺平了道路。

### 2. 性能提升

- 在石英玻璃上实现了4.8TB的可用容量（301层）
- 在硼硅酸盐玻璃上实现了2.02TB

### 3. 硬件简化

引入"相位体素"（phase voxels）技术，只需单脉冲即可写入，且读取设备从需要3-4个摄像头简化为只需1个摄像头。

### 4. 生命周期

加速老化实验表明数据在室温下可保存超过10,000年。

## 六、关键技术信息

### 1. 技术演进里程碑

**2026年突破**：
- 介质商用化转型：成功引入硼硅酸盐玻璃，大幅降低耗材成本
- 相位体素技术：实现单脉冲写入，每比特功耗降至8.85-10.1nJ
- 硬件架构极简优化：读取端硬件精简为单摄像头架构
- 吞吐量性能跨越：通过4光束独立调制并行写入，吞吐量提升至65.9Mbit/s
- 极端耐用性验证：基于Arrhenius law加速老化测试，证实数据在290°C高温下仍保持稳定

### 2. 底层物理原理与介质特性

Project Silica通过飞秒激光（脉冲时间仅为千万亿分之一秒）在透明介质内部引发永久性物理结构改变，形成三维体素。

### 3. 介质规格与物理稳定性

- 材料与尺寸：标准盘片为120mm见方、2mm厚的玻璃板
- 支持廉价的硼硅酸盐玻璃（如厨用Pyrex）及高性能合成石英玻璃
- 环境零依赖：无需磁带库所需的精密温湿度控制，在常温环境下可实现零能耗存放，寿命逾10,000年
- 极端韧性测试：经受煮沸、烘烤、微波加热及钢丝绒擦拭后，内部数据仍完好

### 4. 核心写入技术

- 体素生成原理：激光束聚焦在玻璃内部产生等离子体，形成3D数据点
- 写入方案细节：利用各向同性折射率（RI）变化，支持匹配激光最大重复频率（10MHz）的单脉冲写入
- 2mm厚度内可叠加多达258层至301层数据

### 5. 读取与解码架构

- 光学寻址：利用宽视野光学显微镜进行随机访问
- 读取头可直接定位至X/Y/Z轴任意点，彻底消除磁带的寻址延迟
- 通过卷积神经网络(CNN)替代传统复杂硬件来提取数据比特

## 七、优势特点总结

| 特性 | 说明 |
|------|------|
| 超长寿命 | 室温下可保存超过10,000年 |
| 零能耗存放 | 无需温湿度控制，常温零能耗 |
| 高密度 | 4.8TB/石英玻璃，2.02TB/硼硅酸盐玻璃 |
| 快速读取 | 随机访问，无需顺序读取 |
| 高耐久性 | 耐高温、耐水、耐磨损、抗磁脉冲 |
| WORM特性 | 一次写入，多次读取，数据物理锁定 |

## 八、行业意义

Project Silica标志着存储技术进入新纪元，玻璃存储有望成为数据中心和文化遗产保护的革命性解决方案。

微软宣布Silica项目的研究阶段现已正式完成，并将研究成果公开供业界进一步开发。

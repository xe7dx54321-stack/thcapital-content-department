# 聚变新能发布场景机会：锚定核聚变产业化

> 整理日期：2026-03-10
> 标签：#核聚变 #聚变新能 #合肥 #产业化 #2026-03

---

## 一、背景与发布

### 1. 发布概况

2026年2月中旬，在"十五五"开局之年，合肥市场景应用创新促进中心有限公司正式发布《锚定核聚变产业化，聚变新能（安徽）有限公司场景机会发布》。

### 2. 核心主体

作为合肥未来大科学城"聚变能源"赛道的核心市场主体，聚变新能（安徽）有限公司依托**紧凑型聚变能实验装置（BEST）**的工程化建设需求，开放了**4项高能级"硬核"场景机会**。

### 3. 核心目标

加速核聚变技术从**实验室验证迈向工程化应用与商业化落地**。

本次发布聚焦BEST装置建设中的**"卡脖子"工程痛点**，涉及以下四大关键系统：
1. 辐射安全
2. 真空系统
3. 大功率电源
4. 低杂波加热

---

## 二、合作单位

聚变新能作为**中国科学院合肥物质科学研究院等离子体物理研究所磁约束核聚变领域的成果转化平台**，此次开放场景不仅是技术采购需求，更是构建"科研院所+商业公司+场景验证"协同创新生态的关键一步。

---

## 三、四大场景机会详解

### 1. 中子与γ射线剂量率在线实时监测系统

| 项目 | 内容 |
|------|------|
| **机会名称** | 中子与γ射线剂量率在线实时监测系统设计及应用场景 |
| **核心问题** | 中子与γ射线剂量率无法精准在线实时监测、测量误差大、γ射线干扰强及应急响应不足 |
| **技术需求** | 应用中子/γ射线专用探测，高速数据处理、通讯传输及预警算法等技术 |
| **预期目标** | 实现剂量率实时采集、精准测量，数据上传及超阈值报警功能，达到满足辐射防护合规监管，保障人员安全，提升监测效率的目标 |
| **合作方式** | 测试验证、联合开发、应用采购 |

### 2. 真空系统气体注入系统

| 项目 | 内容 |
|------|------|
| **机会名称** | 真空系统气体注入系统设计制造及应用场景 |
| **核心问题** | BEST真空系统气体注入流量波动、气体残留/交叉污染，及难以适配超高真空环境、人工协同差、气体利用率低 |
| **技术需求** | 应用精准流量温控，独立气路、闭环控制及无接触密封技术 |
| **预期目标** | 实现气体精准稳定注入、杜绝污染残留，提升气体利用率的功能，达到保障系统超高真空环境稳定、支撑聚变反应高效安全运行、降低运维成本 |
| **合作方式** | 测试验证、联合开发、应用采购 |

### 3. 固态换流开关（SNU）

| 项目 | 内容 |
|------|------|
| **机会名称** | 固态换流开关（SNU）设计制造及应用场景 |
| **核心问题** | 传统开关响应慢，有电弧损耗、寿命短，且无法适配高压大电流场景快速换向与精准控制 |
| **技术需求** | 应用固态换流开关（SNU）核心技术（含全控型电力电子器件、混合换流拓扑及精准同步控制策略等） |
| **预期目标** | 实现高压大电流下无弧快速切换、故障快速隔离功能，达到缩短切换时间、降低设备损耗、延长使用寿命，提升系统运行稳定性 |
| **合作方式** | 测试验证、联合开发、应用采购 |

### 4. 低杂波加热系统调速管系统

| 项目 | 内容 |
|------|------|
| **机会名称** | 低杂波加热系统调速管系统设计制造及应用场景 |
| **核心问题** | BEST项目低杂波加热系统调速管系统电源不稳定、反馈调节滞后、故障频发及维护繁琐 |
| **技术需求** | 应用高频反馈调节、隔直器优化及智能化监控保护等技术 |
| **预期目标** | 实现调速管系统稳定运行、功率输出精准调控的功能，达到降低故障发生率、简化维护流程，保障系统高效适配BEST项目运行的目标 |
| **合作方式** | 测试验证、联合开发、应用采购 |

## 六、行业意义

### 1. 产业化进程

本次发布标志着核聚变技术从实验室验证迈向**工程化应用与商业化落地**的重要一步。

### 2. 协同创新生态

通过开放四大"卡脖子"场景，构建"科研院所+商业公司+场景验证"**协同创新生态**。

合肥作为中国核聚变研究的重要基地，此次场景发布为全国其他地区的核聚变产业化提供了**可借鉴的模式**。

# 同行资本内容部门

这是同行资本内容部门的内容生产与素材管理工程仓库。仓库保留可复用的业务规范、生产脚本、控制台入口、核心内容资产和活跃生产系统数据；不把本机缓存、运行日志、备份归档和大规模抓取原始素材纳入版本控制。

## 目录结构

- `内容工厂控制台/`：本地内容工厂展示台启动、停止、状态检查入口。
- `内容生产系统/`：较早一版内容生产系统，包含规划、选题、草稿、发布队列、脚本和模板。
- `同行资本市场内容系统/`：从 `/Users/apple/Documents/同行资本市场内容系统` 纳入的活跃市场内容系统。
- `内容素材库/`：精选素材、入池内容、分析归档和少量人工 URL 抓取结果。

## 不入库内容

- macOS/编辑器缓存：`.DS_Store`、IDE 状态、临时文件。
- 运行态文件：日志、PID、控制台 runtime 状态、本地数据库。
- 构建/缓存产物：`__pycache__`、`.pyc`、`node_modules`、前端构建目录。
- 备份归档：`_archive/`、`内容素材库/备份/`、排练备份、压缩包。
- 大规模原始抓取素材：微信抓取、URL 抓取批量原始目录、自动化抓取缓存。
- 旧版 `内容生产系统` 的批量历史产物：雷达原始包、深度抓取正文、旧草稿包、旧发布队列、旧性能数据、旧日志目录。

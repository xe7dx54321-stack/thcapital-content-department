## 从 20 篇样本里稳定看到的文本动作

- 对象、动作、收益同步落地。
- 把复杂概念翻成用户能直接操作的流程。
- 方法论和具体操作之间切换自然。
- 允许有限的场景或自述做入口，但不能让作者表演盖过事实。
- 段落要明显偏短，关键判断句和风险句应独立成段。

## 适合借用的题型

- workflow 拆解
- skill / agent / builder 方法论
- 产品与流程级文章

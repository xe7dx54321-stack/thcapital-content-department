# 文章提及 Skill 分类清单（完整版）

## 文档说明

这份文档基于你提供的文章《内容创作者 7 大热门 Skills 推荐》，并进一步把文中提到的两个技能包：

- `baoyu-skills`
- `Marketing Skills for AI Agents`

里面包含的**所有子 skill 全部展开**。

这次只做 3 件事：

1. 按类别分类

3. 用简短的话说明它是做什么的、效果大概怎样

**说明**：
- 本文档是为了方便你决策，不做“该不该用”的判断。

## 一、文章里直接提到的单独 Skill

这一部分是文章中单独点名的 skill，不属于技能包展开项。

### 1. Agent Reach

- **负责什么**：跨多个平台搜索和读取内容，包括小红书、X、YouTube、Reddit、B站、RSS 等。
- **效果怎样**：适合围绕某个主题做全网摸排，把不同平台上的讨论点、争议点、表达角度一起捞回来。

### 2. Youtube-ultimate

- **负责什么**：搜索 YouTube 视频、提取字幕、分析频道、下载视频内容。
- **效果怎样**：适合把海外视频、发布会、访谈快速转成可引用、可整理的文本素材。

### 3. Humanizer

- **负责什么**：把已有文本改得更自然、更像真人写的，降低 AI 味。
- **效果怎样**：适合在稿件基本成型后再做一轮语言润色，不改核心判断，主要提升可读性和人味。

### 4. description（ppt-generator）

- **负责什么**：把讲稿转成乔布斯风、极简科技感的 HTML 演示稿。
- **效果怎样**：适合把长文或讲稿快速改造成适合分享、路演、汇报的展示版本。

### 5. frontend-slides

- **负责什么**：把内容做成更完整、更动态的 HTML Slides。
- **效果怎样**：适合做更丰富的演示、培训或逐页展示内容。

## 二、`baoyu-skills` 技能包完整展开（18 个子 Skill）

这个技能包更像“内容成品化工具箱”，覆盖网页转 markdown、视频字幕、配图、信息图、图文发布、翻译等内容生产后半段动作。

## 2.1 采集 / 转换 / 内容整理类

### 1. `baoyu-url-to-markdown`

- **负责什么**：把任意网页抓下来并转成 markdown，支持动态网页和 HTML 快照保存。
- **效果怎样**：适合把网页文章、资料页、博客内容沉淀成可编辑、可归档的文本素材。

### 2. `baoyu-youtube-transcript`

- **负责什么**：下载 YouTube 字幕/转录、封面、元数据，支持多语言、章节和翻译。
- **效果怎样**：适合把长视频快速转成文字材料，便于摘录和继续写作。

### 3. `baoyu-danger-x-to-markdown`

- **负责什么**：把 X / Twitter 的帖子和长文转成 markdown，并带 front matter。
- **效果怎样**：适合把 X 上的重要推文串、长文内容沉淀成结构化文档。
- **特别说明**：名字里带 `danger`，意味着它依赖逆向 API，使用时要额外关注稳定性和风险。

### 4. `baoyu-translate`

- **负责什么**：做多语言文章/文档翻译，支持快速、普通、精修三种模式。
- **效果怎样**：适合把海外资料、英文长文、跨语种内容快速转成可用版本。

### 5. `baoyu-format-markdown`

- **负责什么**：把普通文本或 markdown 重新整理成更规范的 markdown 结构。
- **效果怎样**：适合清洗草稿、整理文章结构、补齐标题、摘要、层级、列表等格式。

### 6. `baoyu-markdown-to-html`

- **负责什么**：把 markdown 转成带样式的 HTML，兼容微信公众号风格。
- **效果怎样**：适合把已经写好的 markdown 稿件进一步转成适合发布或展示的 HTML 成品。

## 2.2 视觉生成 / 图文成品类

### 7. `baoyu-article-illustrator`

- **负责什么**：分析文章结构，识别适合配图的位置，并生成对应插图。
- **效果怎样**：适合给长文补视觉辅助，让文章不只是纯文字堆叠。

### 8. `baoyu-cover-image`

- **负责什么**：生成文章封面图，支持不同配色、风格、情绪和比例。
- **效果怎样**：适合做公众号封面、文章头图、专题封面。

### 9. `baoyu-image-gen`

- **负责什么**：通用 AI 图像生成，支持多家图像模型和多种比例、参考图方式。
- **效果怎样**：适合在内容创作中补各种通用配图、概念图、风格图。

### 10. `baoyu-infographic`

- **负责什么**：生成专业信息图，支持多种 layout 和 visual style。
- **效果怎样**：适合把文章、报告、知识点做成高密度信息图，便于传播和展示。

### 11. `baoyu-xhs-images`

- **负责什么**：把内容拆成适合小红书的多张图文卡片，支持多种风格和排版。
- **效果怎样**：适合把一篇长内容快速改造成小红书图卡系列。

### 12. `baoyu-slide-deck`

- **负责什么**：把内容生成成专业 slide deck 图像，并可进一步合并成 PPT/PDF。
- **效果怎样**：适合把文章、分享稿快速改造成演示文稿资产。

### 13. `baoyu-comic`

- **负责什么**：生成知识漫画、教育漫画、教程漫画等序列化漫画内容。
- **效果怎样**：适合把复杂知识做成更轻、更有传播感的视觉形式。

## 2.3 发布 / 分发类

### 14. `baoyu-post-to-wechat`

- **负责什么**：把内容发到微信公众号，支持文章、图文、API 或浏览器方式。
- **效果怎样**：适合把已完成的稿件直接推进到公众号发布链路里。

### 15. `baoyu-post-to-weibo`

- **负责什么**：把内容发到微博，支持普通帖子和头条文章。
- **效果怎样**：适合把内容进一步适配到微博分发场景。

### 16. `baoyu-post-to-x`

- **负责什么**：把内容发到 X / Twitter，支持普通帖子和长文。
- **效果怎样**：适合把内容同步到 X，适配海外社交分发。

## 2.4 辅助工具 / 实验类

### 17. `baoyu-compress-image`

- **负责什么**：压缩图片，转成 WebP 或 PNG，减小文件体积。
- **效果怎样**：适合做发布前图片优化，减少加载压力。

### 18. `baoyu-danger-gemini-web`

- **负责什么**：通过逆向 Gemini Web API 生成文本和图片，也支持视觉输入和多轮对话。
- **效果怎样**：更像一个实验型生成能力补充，可作为额外模型工具使用。
- **特别说明**：名字里带 `danger`，意味着更偏实验/非官方路径，稳定性和安全性要单独评估。

## 三、`Marketing Skills for AI Agents` 技能包完整展开（33 个子 Skill）

这个技能包更像“营销方法论工具库”，覆盖内容策略、SEO、社媒、广告、CRO、生命周期、销售支持等多个营销环节。

## 3.1 品牌上下文 / 内容策略类

### 1. `product-marketing-context`

- **负责什么**：建立产品/品牌的营销上下文文档，包括定位、受众、差异化、语气等。
- **效果怎样**：适合给后续所有营销类 skill 提供统一背景，避免每次都从零解释业务。

### 2. `content-strategy`

- **负责什么**：规划内容策略、内容支柱、选题方向、内容路线图。
- **效果怎样**：适合解决“接下来该写什么、围绕什么长期做内容”的问题。

### 3. `marketing-ideas`

- **负责什么**：提供营销创意、增长点子和推广方向。
- **效果怎样**：适合在灵感不足时做思路发散，补想法池。

### 4. `launch-strategy`

- **负责什么**：规划产品发布、功能上线、活动上线的传播策略。
- **效果怎样**：适合做一次新品/新功能/新活动上线时的传播节奏设计。

### 5. `free-tool-strategy`

- **负责什么**：规划“免费工具”型营销策略，用工具换流量、品牌和线索。
- **效果怎样**：适合做免费小工具获客、内容带流量等打法。

### 6. `marketing-psychology`

- **负责什么**：把心理学、行为科学、认知偏差应用到营销表达和转化设计中。
- **效果怎样**：适合提升内容和营销动作的说服力。

## 3.2 内容创作 / 社媒表达类

### 7. `social-content`

- **负责什么**：创建、安排和优化社交媒体内容，覆盖多个社交平台。
- **效果怎样**：适合做社媒内容规划、多平台重写、内容日历安排。

### 8. `copywriting`

- **负责什么**：写或重写营销文案，如首页、落地页、功能页、定价页等。
- **效果怎样**：适合做正式营销文案创作，而不是日常随手表达。

### 9. `copy-editing`

- **负责什么**：审校和优化已有营销文案，做多轮编辑 sweep。
- **效果怎样**：适合在已有文案基础上，进一步提高清晰度、语气、说服力和具体性。

### 10. `lead-magnets`

- **负责什么**：设计或优化用于获客的 lead magnet，如资料包、清单、模板等。
- **效果怎样**：适合用内容资产换线索和邮箱。

### 11. `email-sequence`

- **负责什么**：创建邮件序列、drip campaign、自动化邮件流。
- **效果怎样**：适合做线索培育、用户跟进和生命周期触达。

### 12. `cold-email`

- **负责什么**：写 B2B 冷启动邮件和后续跟进邮件。
- **效果怎样**：适合做主动外联、BD、销售开发等场景。

### 13. `ad-creative`

- **负责什么**：批量生成广告创意文案，如标题、描述、主文案等。
- **效果怎样**：适合广告投放时快速扩展创意版本。

## 3.3 SEO / 搜索发现类

### 14. `ai-seo`

- **负责什么**：优化内容在 AI 搜索、AI 答案、LLM 引用中的可见性。
- **效果怎样**：适合面向 ChatGPT、Perplexity、AI Overviews 这类新搜索入口做优化。

### 15. `seo-audit`

- **负责什么**：审查网站 SEO 问题，包括技术 SEO 和页面层问题。
- **效果怎样**：适合查为什么不收录、为什么不排名、SEO 哪些地方有缺陷。

### 16. `programmatic-seo`

- **负责什么**：用模板和数据批量生成 SEO 页面。
- **效果怎样**：适合目录页、模板页、规模化 SEO 运营。

### 17. `schema-markup`

- **负责什么**：添加或优化 schema markup / structured data。
- **效果怎样**：适合提升搜索结果中的结构化展示和富结果机会。

### 18. `site-architecture`

- **负责什么**：规划网站信息架构、导航、URL 结构和内部链接。
- **效果怎样**：适合做站点结构梳理，让内容体系更清晰、更利于 SEO。

### 19. `competitor-alternatives`

- **负责什么**：创建竞争对手对比页和 alternatives 页面。
- **效果怎样**：适合兼顾 SEO 和销售支持，把用户从竞品比较场景中截流过来。

## 3.4 转化率优化（CRO）类

### 20. `page-cro`

- **负责什么**：优化首页、落地页、功能页等页面的转化表现。
- **效果怎样**：适合提升营销页面的点击、注册、咨询等结果。

### 21. `form-cro`

- **负责什么**：优化各类表单的转化率，如联系表单、申请表单、demo 请求表单。
- **效果怎样**：适合减少表单流失、提升填写完成率。

### 22. `signup-flow-cro`

- **负责什么**：优化注册、注册引导、试用激活等流程。
- **效果怎样**：适合减少注册摩擦，提升注册到激活转化。

### 23. `onboarding-cro`

- **负责什么**：优化注册后的 onboarding 和首次使用体验。
- **效果怎样**：适合提升激活率、缩短 time-to-value。

### 24. `popup-cro`

- **负责什么**：优化弹窗、模态框、浮层、退出意图提示等。
- **效果怎样**：适合用弹窗提升转化，但同时控制打扰程度。

### 25. `paywall-upgrade-cro`

- **负责什么**：优化 paywall、升级页、付费弹窗、功能解锁页。
- **效果怎样**：适合提升从免费到付费、从低档到高档的升级转化。

### 26. `ab-test-setup`

- **负责什么**：设计和规划 A/B test，包括假设、实验设计和测试时长。
- **效果怎样**：适合在多个版本之间做可验证的实验，而不是拍脑袋改动。

## 3.5 广告 / 流量 / 分析类

### 27. `paid-ads`

- **负责什么**：规划和优化 Google、Meta、LinkedIn、X 等平台的广告投放。
- **效果怎样**：适合做付费流量获取和广告策略设计。

### 28. `analytics-tracking`

- **负责什么**：设置或审查分析埋点、事件追踪、转化追踪、GA4、GTM 等。
- **效果怎样**：适合解决“怎么衡量有没有效果”的问题。

### 29. `pricing-strategy`

- **负责什么**：规划定价、套餐设计、商业模式和 monetization 策略。
- **效果怎样**：适合处理收费体系、分层策略和定价结构问题。

### 30. `referral-program`

- **负责什么**：设计和优化推荐计划、裂变机制、affiliate 或 ambassador 体系。
- **效果怎样**：适合做口碑传播和用户带用户的增长打法。

## 3.6 生命周期 / 收入协同 / 销售支持类

### 31. `churn-prevention`

- **负责什么**：降低流失、设计取消挽回流程、保留策略和 retention 机制。
- **效果怎样**：适合处理用户留存和减少退订。

### 32. `revops`

- **负责什么**：处理 revenue operations、线索流转、营销到销售的衔接流程。
- **效果怎样**：适合打通营销、销售、收入管理之间的协同机制。

### 33. `sales-enablement`

- **负责什么**：制作销售资料，如 pitch deck、one-pager、异议处理文档、demo script。
- **效果怎样**：适合帮助销售团队更高效地拿着内容去推进成交。

## 四、把文章涉及的所有 Skill 再按工作流理解一遍

如果从“内容工作到底怎么流动”这个角度看，可以这样理解：

### 1. 先找信息 / 抓原始素材

- `Agent Reach`
- `Youtube-ultimate`
- `baoyu-url-to-markdown`
- `baoyu-youtube-transcript`
- `baoyu-danger-x-to-markdown`

作用：先把分散在平台上的内容、网页、视频、社交帖文变成可消费的原始素材。

### 2. 再整理文字 / 跨语种处理

- `Humanizer`
- `baoyu-translate`
- `baoyu-format-markdown`
- `baoyu-markdown-to-html`
- `copy-editing`
- `copywriting`
- `social-content`

作用：把原始素材整理、改写、润色成更适合发布的文本。

### 3. 再做视觉成品

- `baoyu-article-illustrator`
- `baoyu-cover-image`
- `baoyu-image-gen`
- `baoyu-infographic`
- `baoyu-xhs-images`
- `baoyu-slide-deck`
- `baoyu-comic`
- `description / ppt-generator`
- `frontend-slides`

作用：把内容从文字进一步变成图文卡片、信息图、封面图、演示稿、漫画等更适合传播的形式。

### 4. 再做发布与分发

- `baoyu-post-to-wechat`
- `baoyu-post-to-weibo`
- `baoyu-post-to-x`

作用：把已经完成的内容推到具体平台去。

### 5. 再做策略、增长与优化

- `product-marketing-context`
- `content-strategy`
- `marketing-ideas`
- `launch-strategy`
- `lead-magnets`
- `ai-seo`
- `seo-audit`
- `programmatic-seo`
- `page-cro`
- `ab-test-setup`
- `analytics-tracking`
- `paid-ads`
- `churn-prevention`
- `revops`
- `sales-enablement`
- 以及其他 CRO / pricing / referral 类 skill

作用：把内容从“写出来”进一步推进到“能带来增长、线索、转化和收入协同”。

所以这次已经把两个技能包：

- `baoyu-skills` 的 **18 个子 skill**

全部拉平展开，并且都做了简要解释。

这样你接下来就可以很直观地看：

1. 哪些是搜索采集类
2. 哪些是写作与改写类
3. 哪些是视觉成品类
4. 哪些是发布类
5. 哪些是营销增长 / SEO / CRO / 广告 / 销售支持类

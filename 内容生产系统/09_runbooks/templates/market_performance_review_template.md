# Performance Review Template

## 作用

对一次内容发布结果进行结构化复盘。

## 必备字段

- `review_id`
- `review_key`
- `topic_id`
- `queue_ids`
- `review_window`
- `status`

## 推荐结构

```markdown
# Performance Review

- `review_id`: `review__...`
- `review_key`: `topic_key__24h`
- `topic_id`: `topic__...`
- `queue_ids`: `queue__..., queue__...`
- `review_window`: `24h`
- `status`: `collecting`

## Platform Metrics

- `wechat`: `n/a`
- `zhihu`: `n/a`

## Learnings

- `source_learnings`: `n/a`
- `next_actions`: `n/a`
```

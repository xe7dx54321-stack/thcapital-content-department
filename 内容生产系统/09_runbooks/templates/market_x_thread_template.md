# X Thread Template

## 角色

- 全球信号雷达
- 负责“快速发信号 + 快速给判断”

## 推荐结构

1. Hook：一句话结论
2. 1 句背景桥接（如对象不够显而易见）
3. 2-5 条拆解
4. 至少 1 条给 why-you-should-care
5. 最后一条给 follow-up angle / 问题 / 互动

## 语气

- 短
- 准
- 每条都要有信息增量

## 最低要求

- 不得把中文长文直接压缩搬上去
- 一次 thread 只服务一个主要受众层
- thread 中要保留一个最关键 source ref
- 第一条与第二条只能承载短判断 / 短背景，不要塞完整 approved angle
- 能一眼看懂你在判断什么
- 第一行必须可单独截图传播
- 每条都必须有信息增量

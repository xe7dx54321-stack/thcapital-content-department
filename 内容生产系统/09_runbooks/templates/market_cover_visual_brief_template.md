# Cover Visual Brief Template

## 目标

为长文 / 多平台内容提供统一视觉 brief。

## 必须回答

1. 这张图是为了让用户点开，还是为了让用户快速看懂？
2. 用户 3 秒内能不能知道对象是什么？
3. 用户 3 秒内能不能知道为什么现在值得看？
4. 这张图会不会把正文真正的主张带歪？

## 推荐内容

- 平台 / 使用场景
- 目标读者
- 封面任务
- 核心标题
- 副标题候选
- 首屏承诺
- 一句话背景 cash line
- 核心判断
- 封面主 claim
- 不应误导用户的点
- 视觉构图建议
- 封面图建议
- 插图建议
- 适合引用的 key message
- 人工 QC 清单

# WeChat HTML Handoff Template

## 目标

把公众号稿整理成更适合后续 HTML 化与发布排版的 handoff 说明。

## 推荐内容

- 主标题
- 副标题 / 导语建议
- 首屏结构建议
- 前 3 屏分别讲什么
- 第一处 proof anchor 放在哪里
- 第一处“为什么你该关心”放在哪里
- 一级 / 二级小标题结构
- 哪些段落适合加强调
- 哪些句子适合做引用框 / 金句框
- 哪些位置适合插图 / 引用框
- 哪些内容必须保留 source refs
- 仍需人工完成的排版动作

## 注意

- 这里只写 handoff，不声称 HTML 已经完成
- 首屏不能只是长段文字
- 至少要标出 1 个 quote block 和 1 个可视化位置

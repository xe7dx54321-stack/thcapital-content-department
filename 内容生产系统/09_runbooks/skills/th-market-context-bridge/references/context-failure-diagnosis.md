# Context Failure Diagnosis

Use this file to determine whether a weak piece is suffering from a context problem.

## 1. Symptoms of missing context

- readers click but comment “这是什么？”
- the opening feels sharp but confusing
- the argument starts before the object is concrete
- users understand isolated sentences but not the whole piece

## 2. Symptoms of too much context

- readers feel the article starts too slowly
- the opening reads like a news brief
- the main judgment comes too late
- the first screen contains history the reader does not need

## 3. Fix order

Repair in this order:

1. define the missing object
2. shorten the event description
3. add one line explaining why this matters now
4. bridge directly into the main claim

## 4. Learning output

Every context diagnosis should try to say:

- what the reader likely failed to understand
- which missing fact caused the confusion
- whether the fix is:
  - more context
  - earlier context
  - shorter context
  - clearer translation

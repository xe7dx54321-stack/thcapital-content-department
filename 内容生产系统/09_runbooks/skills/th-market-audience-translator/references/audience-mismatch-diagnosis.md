# Audience Mismatch Diagnosis

Use this file when a draft feels “not for anyone in particular”.

## 1. Symptoms

### A. Too insider

- object assumed, not explained
- too many terms too early
- reader does not know why to care

### B. Too fluffy

- jargon removed, but nothing substantive remains
- sounds motivational instead of analytical

### C. Too broad

- tries to speak to everyone
- payoff keeps changing
- the piece never lands for one clear reader

### D. Too abstract for platform

- especially common on Xiaohongshu and X
- the copy starts from framework instead of user stake

## 2. Repair order

1. choose one primary layer
2. restate the reader’s core question
3. reorder the explanation around that question
4. translate terminology
5. recalculate the payoff frame

## 3. Learning output

Every diagnosis should try to state:

- who the draft accidentally served
- who it should serve
- what line or section reveals the mismatch

# Hook Library

Use this file to choose the **one main hook family** for a TH Capital content package.

Do not stack multiple families unless the platform clearly supports it.

## 1. Result-first

### When to use

- when the reader wants the takeaway fast
- when the conclusion is sharper than the event itself
- when the strongest value is “what this means”

### Formula

- `真正值得看的，不是 X，而是 Y`
- `这不是 X，它其实在说明 Y`
- `真正能赚钱 / 能落地 / 能扩散的，不是 A，而是 B`

### Best for

- WeChat
- Zhihu
- X

---

## 2. Anti-consensus

### When to use

- when public interpretation is shallow
- when TH Capital has a sharper view than generic media
- when the topic is being widely misunderstood

### Formula

- `大家都以为这是 X，但更重要的是 Y`
- `这件事最容易看错的地方，是 X`
- `看起来像 X，实际上它在改变 Y`

### Best for

- WeChat
- Zhihu
- X

---

## 3. User-stake

### When to use

- when the reader needs to feel “this affects me”
- when the topic is abstract unless translated into workflow / opportunity / personal leverage

### Formula

- `这件事表面和你无关，但它会影响你以后怎么 X`
- `如果你在关注 X，这件事值得立刻看`
- `对普通人 / 创作者 / 小团队来说，这件事真正重要的是 Y`

### Best for

- Xiaohongshu
- WeChat

---

## 4. Structural-shift

### When to use

- when a platform / stack / distribution layer is changing
- when the story is bigger than one product release

### Formula

- `这不是功能更新，而是入口开始换人`
- `这件事说明行业开始从 X 走向 Y`
- `关键变化不是产品本身，而是底层结构变了`

### Best for

- WeChat
- Zhihu
- X

---

## 5. Risk-warning

### When to use

- when the reader is complacent
- when the topic exposes a hidden failure mode
- when the right frame is “you are underestimating the downside”

### Formula

- `真正危险的不是 X，而是 Y`
- `大家没意识到，这条链最脆弱的地方在 X`
- `越成熟的系统，越可能死在这个小地方`

### Best for

- WeChat
- X
- Zhihu

---

## 6. Opportunity-window

### When to use

- when the topic implies a narrow timing edge
- when the question is not “is this real” but “whether to act now”

### Formula

- `这条线如果现在不盯，半年后大概率会错过`
- `真正有价值的，不是热度本身，而是它提前暴露了 X`
- `这不是终局，但很可能是窗口期开始`

### Best for

- WeChat
- Xiaohongshu
- X

---

## 7. Selection rules

Choose hooks in this order:

1. What is the strongest truthful claim?
2. What is the most likely reader misunderstanding?
3. Which hook family fixes that misunderstanding fastest?

If two hook families both work:

- prefer the one that is easier to understand in one glance
- prefer the one that creates less title/body mismatch

## 8. Prohibited patterns

- vague “风口来了”
- fake urgency
- titles that hide the subject too long
- titles that sound deep but explain nothing
- hooks that only work if the reader already knows the event

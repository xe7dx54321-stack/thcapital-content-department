# Postmortem Dimensions

Use these dimensions to analyze published content in TH Capital’s content factory.

## 1. Click layer

Questions:

- Did users click?
- Was the title clear and attractive?
- Did the cover communicate the right stake?

Likely related components:

- hook / title / cover
- timing

## 2. Cold-start clarity layer

Questions:

- After clicking, did users quickly understand the object and event?
- Was the context bridge early and clear enough?

Likely related components:

- context bridge
- opening structure

## 3. Read-through layer

Questions:

- Did the body sustain attention?
- Were paragraphs, subheads, and rhythm suitable for the platform?

Likely related components:

- platform renderer
- polish

## 4. Audience-fit layer

Questions:

- Did the piece answer the right user question?
- Was the payoff framed for the intended reader?

Likely related components:

- audience translator
- platform selection

## 5. Conversion / follow-up layer

Questions:

- Did the piece create the desired follow-up?
- Was the CTA proportionate and clear?

Likely related components:

- platform renderer
- brand context
- topic choice

# th-content-repurpose

Use this skill for Step 7 basic productization work:

- WeChat HTML handoff planning
- cover / visual brief generation
- Xiaohongshu card brief generation
- light repurposing guidance

## Important boundary

- This skill is for **handoff assets**, not full auto-publishing.
- It should work off a polished Draft Pack.
- It should preserve the same core judgment as the source draft.

Its job is:

> 把已经打磨过的内容，进一步整理成“更接近可发成品”的交付说明。

## Load order

1. Read `../../../08_brand_assets/20260324_平台定位与表达规范.md`
2. Read `../th-market-platform-renderer/SKILL.md`
3. Read `../../../09_runbooks/20260325__market-content-polish-runbook.md`
4. If available, read `../../../08_brand_assets/latest__head-media-learning-rulebook-v1.md`
5. Read `../../../09_runbooks/templates/market_wechat_html_handoff_template.md`
6. Read `../../../09_runbooks/templates/market_cover_visual_brief_template.md`
7. Read `../../../09_runbooks/templates/market_xiaohongshu_card_brief_template.md`
8. Read the selected Draft Pack

## Core workflow

1. Keep the approved angle fixed.
2. Turn the content into handoff-ready assets:
   - WeChat formatting guidance
   - cover / visual brief
   - Xiaohongshu card decomposition brief
   - renderer-aware final structure hints
3. Keep citations and risk note visible where relevant.
4. Do not claim image production or HTML conversion has already happened unless it really has.

## Output guidance

Output:

1. WeChat HTML handoff
2. cover / visual brief
3. Xiaohongshu card brief if relevant
4. repurpose notes / remaining gaps

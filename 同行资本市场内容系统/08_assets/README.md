# Phase 17 Image Asset Workspace

This directory stores metadata for manual image generation tasks and image asset library records.

`images/` is reserved for human-generated, manually uploaded, screenshot, or design-produced image files. Real image files are local runtime assets and must not be committed to Git.

Phase 17 rules:

- No automatic image generation.
- No image model calls by default.
- No unknown-copyright image downloads.
- Image usage requires human visual review.
- Article publishing remains manual.

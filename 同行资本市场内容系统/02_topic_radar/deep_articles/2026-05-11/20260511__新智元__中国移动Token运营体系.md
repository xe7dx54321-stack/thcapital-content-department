##### **【新智元导读】中国移动发布Token运营体系，不是简单卖AI算力，而是要把网络、算力、模型、应用和10亿用户连接起来，打造AI时代的「新梦网」。**

2026 年 3 月，国家数据局局长刘烈宏在中国发展高层论坛上扔出了一颗「数据炸弹」——
中国日均 Token 调用量已突破 140 万亿——两年增长超过 1000 倍！

![Image 5](https://simg.baai.ac.cn/hub-detail/fda71d0f7bd1563168051911e64dd2d71778397602096.webp)

2024 年初，这个数字还只是 1000 亿。短短两年，Token 消耗量走出了一条近乎垂直的增长曲线。

驱动这条曲线的，不再是「问 ChatGPT 一个脑筋急转弯」这样的轻量对话。

从 OpenClaw「全民养虾」到 Hermes Agent「全民养马」，AI 智能体正在变成 24 小时不休息的「数字员工」——单个智能体完成一次任务，消耗的 Token 是传统 AI 对话的数十倍。

而 AI 视频生成更是 Token 消耗的「超级黑洞」：生成 1 分钟 720P 视频，平均要烧掉超过百万 Token，是文本场景的数千倍。

**Token****，正在成为****AI****时代的「水电煤」。**

中国移动在 2026 移动云大会上正式发布了 Token 运营体系，宣布要把 Token 打造成连接算力、模型、应用与用户的「通用货币」。

![Image 6](https://simg.baai.ac.cn/hub-detail/bcbda1b7a6122609d6f958d2e8d4fa831778397602096.webp)
一家年营收超万亿、坐拥 10 亿用户的通信巨头，向全行业宣告：

作为央企 AI 国家队、数字中国建设主力军，中国移动坚决贯彻落实党中央决策部署，全面实施「网络强基、全栈创新」战略，充分依托运营商全网用户、全域渠道、云网算力独有优势，在 Token 产业发展中主动站位、主动作为。

在演讲中，中国移动董事长提出「算力即动能、AI 即服务、开放促共赢」。

**这次不只是要卖****Token****，中国移动要做****AI****时代的「新基建运营商」。**

![Image 7](https://simg.baai.ac.cn/hub-detail/df9d89c9b2ec34282d3fc8aa39faec001778397602096.webp)**一个「管道商」的必然进化**
有人可能会问：Token 这件事，互联网大厂不是早就在做了吗？

OpenRouter、硅基流动这样的平台已经跑通了商业模型，阿里云、腾讯云也在卖 Token 套餐，中国移动凭什么来「搅局」？

这个问题，恰恰问反了。

让我们先看一组数据。

中国移动 2025 年营收 1.05 万亿元，移动客户超 10 亿户，5G 网络基站超 277 万站，智算总规模达 92.5EFLOPS，实现了从百卡到超万卡的全规格算力覆盖。

![Image 8](https://simg.baai.ac.cn/hub-detail/1fc31a7339e1593bc6f01946171eb12b1778397602097.webp)

2026 年，公司算力网络和智能网络资本开支分别增长 62.4%和 19.8%，算力投资占总资本开支的比重已超过 37%。

简单来说，中国移动手里攥着三张别人很难复制的底牌：

**第一张牌：超大规模的算网基础设施。**

万卡级智算中心集群、毫秒级全光高速直连网络、遍布全国的边缘节点——这是 Token 大规模生产和低延迟调度的物理根基。

互联网公司可以做云、做模型，但要把算力「像自来水一样」输送到每个角落，没有网络基础设施的支撑根本做不到。

**第二张牌：****10****亿级用户的触达能力。**

中国移动的用户不是「注册用户」，而是每天实实在在使用手机通信的活跃用户。

这意味着一个现成的、有付费习惯的、覆盖城乡所有人群的超级渠道。

模型厂商最头疼的「分发」问题，在中国移动这里不是问题。

**第三张牌：「云、管、端、边」一体化的运营体系。**

从云端的算力中心到管道里的网络传输，从用户手中的终端到散落各地的边缘节点，中国移动有一套完整的「端到端」体系。

这套体系，经历了从语音到流量到数据的多轮转型，已经被验证过是可规模化运营的。

![Image 9](https://simg.baai.ac.cn/hub-detail/ff91e2c902c6bc66cfbd964fd5889add1778397602098.webp)

这就是为什么中国移动市场经营部总经理邱宝华在发言中说了这样一句话：**当前****AI****产业「不缺单点突破的技术能力，缺的是打通用户与价值的完整链路」；「不缺百花齐放的创新产品，缺的是可持续、可规模化的商业闭环」。**

![Image 10](https://simg.baai.ac.cn/hub-detail/df9d89c9b2ec34282d3fc8aa39faec001778397602098.webp)**AI时代的「新梦网」**
很多人可能不记得「移动梦网」了，但这个诞生于 2000 年代初的商业模式，曾经深刻地改变了中国互联网的命运。

![Image 11](https://simg.baai.ac.cn/hub-detail/2428e9d2a25ec1a4faaa3aebf70bdf281778397602098.webp)

在移动互联网尚未到来的年代，无数 CP（内容提供商）有好内容、有好产品，却苦于找不到用户、收不到钱。

移动梦网做了一件事：用运营商的计费通道和用户通道，把内容和用户连接起来。SP/CP 负责提供内容，中国移动负责分发和结算，用户用话费买单。

这套模式催生了一整个产业：腾讯早期的 QQ 会员增值业务、新浪搜狐的彩信内容、无数游戏和音乐服务商，都是在梦网体系里拿到了「第一桶金」。

今天，AI 产业面临的困境和二十年前何其相似——

模型厂商有好模型，但缺乏高效的分发渠道和商业变现路径；AI 应用厂商有好产品，但缺乏稳定的用户触达入口；生态伙伴想入局，但缺乏统一的身份认证、计量计费和结算体系，运营门槛极高。

![Image 12](https://simg.baai.ac.cn/hub-detail/621422eeabac958bdb97192afbb87b681778397602098.webp)

中国移动要打造的 Token 运营体系，本质上就是 AI 时代的「新梦网」。

但它比梦网更进了一步。

梦网时代连接的是「内容与用户」，Token 运营要连接的是「算力、模型、应用与用户」——个四维的价值网络。

中国移动不是要自己去训练最好的大模型，也不是要自己去做最火的 AI 应用，而是要做 AI 产业的「中间层」：**向上对接算力和模型，向下触达用户和场景，中间提供统一的计费、结算、鉴权和分发平台。**

这套体系的架构非常清晰：

**能力底座层**——升级算力基础设施，汇聚大模型、智能体、Skills 等能力，确保 Token 的高质量供给。

移动云已升级模型服务平台 MoMA，接入了 MiniMax、豆包、GLM 等 200 多款主流模型，通过统一 API 调用实现多模型协同调度，并提供「普惠、精品、机密」三档 Token 服务。

**运营平台层**——建设 Token 统一运营平台，实现统一计费、统一结算、统一鉴权，支撑 Token 在整个生态中高效流转。

这一层是整个体系的心脏，它解决的是 AI 产业最基础、也最棘手的问题：怎么量、怎么算、怎么分。

**应用入口层**——围绕内外部生态，打造统一的应用入口体系，实现 Token 的规模化消费。

从 12580 通话入口到灵犀智屏终端，从自有产品 Skill 化到与腾讯 WorkBuddy 等外部生态的深度合作，覆盖办公、生活、娱乐的全场景。

![Image 13](https://simg.baai.ac.cn/hub-detail/5c5d8a402ccd1d304261caa3346d39e21778397602098.webp)

有一个非常精准的表达：「Byte+Token」双高速增长。

Byte 是流量时代的计价单位，Token 是智能时代的计价单位。

中国移动要做的，不是用 Token 替换 Byte，而是在 Byte 的基础上叠加 Token，让传统通信业务和 AI 智能服务形成「双轮驱动」。

![Image 14](https://simg.baai.ac.cn/hub-detail/df9d89c9b2ec34282d3fc8aa39faec001778397602098.webp)**通算智融合**
宏大的战略愿景，最终要靠可执行的路径来落地。

中国移动为 Token 运营体系设计了「汇聚、筑基、激活、共赢」四大行动，每一步都直指产业痛点。

**行动一：****Token****汇聚****——****打造「****AI****算力超市」**

Token 的源头是算力和模型。

中国移动的策略是「自建+合作+联营」三条腿走路：

*   闭源模型直连接入，快速丰富供给；

*   开源模型自有部署，确保自主可控；

*   联合厂商协同共建优质闭源模型，适配垂直场景。

依托算网大脑 4.0 和 MoMA 模型服务平台，中国移动要构建一个多元异构的算力生态，通过智能调度实现 Token 资源的最优匹配。

![Image 15](https://simg.baai.ac.cn/hub-detail/846076e9613edec8c4d02a020b1c1e151778397602099.webp)

用通俗的话说，就是不管你想用哪家的模型、哪种类型的算力，这里都能「一站配齐」。

**行动二：平台筑基****——****让****AI****像打电话一样好用**

这是四大行动中最具「运营商基因」的一步。中国移动要做三件事：

**推出****Token****套餐，实现「人人一个****Token****账户」。**

江苏移动 5 元/250 万 Token 的 Lite 体验套餐、北京移动 5.99 元起的算力次包，已经把 AI 算力的使用门槛拉到了「一杯奶茶」的水平。

未来还将推出面向开发者、视频创作者等群体的垂域专属套餐。

**统一****Token****量纲，支持跨应用流转。**

今天你在 A 应用消耗的 Token 和在 B 应用消耗的 Token，可能完全无法互通。

中国移动要联合产业各方，建立统一的换算机制，实现 AI 服务「标准化、可统计、可结算、可运营」。

**打通****Token****鉴权，实现「一次认证、全网通行」。**

依托手机码号，升级统一认证服务（IDaaS），让用户在不同 AI 应用之间无缝切换，不用每次都重新注册、重新登录。

这三件事看起来很「基础」，但恰恰是当前 AI 产业最缺的基础设施。

就像 20 年前没有支付宝，电商就做不起来一样——没有统一的 Token 计量、结算和鉴权体系，AI 产业的规模化商用就只是一句空话。

![Image 16](https://simg.baai.ac.cn/hub-detail/aba39f80683b961fe9e117749b83d5891778397602099.webp)

**行动三：消费激活****——****让****Token****消费融入日常生活**

有了供给和平台，还得有消费。中国移动的思路是「内外兼修」：

对内，把传统通信能力「AI 化」——升级 12580、「1 号键」等通话入口为通信智能体，把自有产品能力 Skill 化开放，拓展灵犀 inside 终端生态，让 AI 手机、家庭大屏、智能穿戴都成为 Token 消费的入口。

对外，与腾讯、荣耀、讯飞、阿里等头部厂商深度合作，汇聚优质 AI 应用接入「通用 Token」。

目前已与腾讯合作推出 WorkBuddy 定制版，为用户提供覆盖办公、生活、娱乐的 AI 助理服务。

这一步的关键词是「无处不在」。

中国移动的目标不是让用户专门打开一个 AI 应用去消耗 Token，而是让 Token 消费像流量消费一样，无感地融入日常生活的每一个角落。

![Image 17](https://simg.baai.ac.cn/hub-detail/9c86371901cbd35c92dc862390f0705f1778397602099.webp)

**行动四：价值共赢****——****百亿资源，万亿市场**

据预测，2030 年中国 Token 调用总量将攀升至 700 万万亿次。面对这片蓝海，中国移动抛出了一份诚意十足的合作方案：

推出「能力采买、收益分成、资源共建」三类合作模式，覆盖算力供给、模型服务、AI 应用、渠道销售全链条。

未来 3—5 年，投入百亿级 Token 生态资源，建设千亿级算力基础设施，目标是携手共创万亿级 AI 产业价值。

![Image 18](https://simg.baai.ac.cn/hub-detail/921c54d88b6a5ed223544c2ee1e59d951778397602099.webp)

这不是喊口号。从财报数据看，中国移动 2025 年智算收入增速达 279%，算力服务收入达 898 亿元。

到「十五五」期末，公司目标是算力服务和智能服务收入翻番。真金白银的投入，已经在路上了。

![Image 19](https://simg.baai.ac.cn/hub-detail/2f2b7a18928d1c76d953a72bbb3b481e1778397602099.webp)

![Image 20](https://simg.baai.ac.cn/hub-detail/df9d89c9b2ec34282d3fc8aa39faec001778397602099.webp)**一场「升维」运动****从卖管道到运营生态**
回过头来看，中国移动的 Token 战略到底在做什么？

不是在做一个新产品，而是在构建一套新的商业操作系统。

在通信时代，运营商的商业模式本质上是「卖管道」——你打电话、上网，运营商按分钟、按流量收费。

管道虽然重要，但附加值有限，最终难逃「价格战」的宿命。

在 AI 时代，中国移动正在尝试一次根本性的「升维」：从卖管道，变成运营生态。

Token 不是管道，而是生态的「血液」——它流经算力、模型、应用、用户的每一个环节，把整个 AI 产业链串联成一个有机整体。

这正是中国移动「网络强基、全栈创新」公司战略的集中体现。

「网络强基」提供了别人无法复制的物理底座，「全栈创新」则是要在这个底座之上，构建一套贯穿全产业链的价值运营体系。

Token 运营，就是把这两者连接在一起的那把钥匙。

更深层来看，这场转型回应的是一个更宏大的命题：中国移动能否从一家传统通信运营商，真正跃迁为「世界一流科技服务企业」？

答案藏在行动里。

当中国移动把「通信服务、算力服务、智能服务」明确为三大主业，当它在集团层面专设算力专项办公室——这家公司已经不是在转型了，而是在「重生」。

**让Token像水电一样，流进千家万户**
二十年前，中国移动用一套梦网模式，帮助初生的中国互联网找到了第一条商业化道路。

二十年后的今天，AI 产业站在了同样的路口：技术突破此起彼伏，百模大战硝烟弥漫，但从「技术狂欢」到「商业闭环」之间，还差一座桥。

这次中国移动要**让****Token****像水电一样融入千行百业、走进千家万户，让****AI****服务真正实现全民普惠、人人可及。**

这不是一个轻松的目标。

要让 10 亿用户真正用上 Token、用好 Token，要让数以万计的模型厂商、应用开发者、内容创作者在这个生态里找到自己的位置、获得应有的回报，需要的不仅是技术能力，更是产业整合的智慧、平台运营的耐心，以及面对万亿级市场时的战略定力。

但历史已经证明过一次：**当庞大的网络基础设施遇上正确的商业模式，奇迹就会发生。**

AI 时代的「新梦网」，已经启程。

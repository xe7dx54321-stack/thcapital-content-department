# P31C-004 Source Fetch Method Playbook Report

`config/acquisition_source_playbooks.yaml` defines safe metadata fetch methods by source. It covers official AI labs, GitHub, Hugging Face, arXiv, YC, TechCrunch, FinSMEs, Product Hunt / Trend Hunt, Reddit, Hacker News, newsletters, Chinese AI media, WeChat metadata, YouTube, X, trend heat, manual URL, and keyword discovery.

Forbidden methods remain blocked: browser login, paywall bypass, full-text scrape, and credentialed fetch.

# P30-003 Migrated Source Topic Candidate Promotion Report

## Goal

Promote only confirmable OpenClaw migrated signals into conservative topic candidates.

## Outputs

- `latest_openclaw_activated_topic_candidates.json`
- `latest_openclaw_activated_topic_candidates.md`
- `latest_openclaw_activated_topic_candidates_board.md`

## Boundary

This step does not promote every migrated signal. Weak lanes do not directly enter the brief pipeline, and no topic candidate is marked as hard evidence.

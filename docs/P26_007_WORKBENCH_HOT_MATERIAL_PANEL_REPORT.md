# P26-007 Workbench Hot Material Panel Report

## Goal

Expose upstream material supply in the workbench so the operator can see whether today has enough fresh, useful material.

## Panel

The panel shows source gaps, expansion plan counts, hot signal lanes, backfill tasks, daily hot material pool, quality gate status, weak supply reasons, and top manual backfill tasks.

## Boundary

The panel is read-only. It does not publish, fetch, or mutate source configuration.

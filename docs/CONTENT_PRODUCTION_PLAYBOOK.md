# Content Production Playbook

## 链路

confirmed topic → topic scoring → main topic selection → brief → outline → draft → multi-agent review → rewrite version → final candidate → workbench。

## 核心原则

- 选题必须有 why now、读者问题、证据强度、叙事潜力和差异化。
- brief 必须列出证据库存、缺口、核心张力和写作风险。
- outline 必须有故事线，每个 section 要有 purpose 和回答的读者问题。
- draft 不能伪造引用、不能写 target price、不能自动发布。
- critic / judge / rewrite 不能绕过质量 gate。
- final candidate 必须人工审阅。

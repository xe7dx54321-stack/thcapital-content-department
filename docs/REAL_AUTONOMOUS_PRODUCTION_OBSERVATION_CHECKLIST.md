# Real Autonomous Production Observation Checklist

## 使用方式

连续观察 1-2 天 Autonomous Runtime 自动产出的工作台结果。每天确认 Runtime、采集、选题、brief、outline、draft、review、final candidate 和质量风险。

## 核心问题

- Runtime 是否持续运行。
- 当天 scheduled jobs 是否触发。
- 是否选出成熟主选题。
- 是否出现 source metadata 标题污染。
- 是否证据不足仍强行写稿。
- final candidate 是否值得人工打开看。
- 是否需要批准 Phase33 calibration proposal。

# P28-006 Workbench Evidence & Topic Promotion Panel Report

Phase 28 adds an evidence and topic promotion panel to the WeChat workbench.

The panel appears in the daily operations and system operations sections. It shows connector evidence packet counts, eligible promotion counts, promoted topic candidates, needs-evidence counts, ready-for-brief bridge items, freshness summary, dedup summary, source origin, and weak-signal indicators.

The panel keeps Phase 28 in the upstream material workflow rather than adding a new broad UI surface.

Command:

```bash
make wechat-workbench
```

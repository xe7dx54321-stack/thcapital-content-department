# P29-007 Workbench Source Migration Panel Report

## Scope

Adds an OpenClaw Source Migration panel to the workbench.

## Visible Metrics

- Source inventory count
- P0/P1 migration candidates
- Blocked and manual-only sources
- Weak signal count
- Hard evidence allowed count
- OpenClaw hot material contribution

## Boundary

Workbench display is operator guidance only. It does not trigger gateway, cron, fetch, publish, or config mutation.

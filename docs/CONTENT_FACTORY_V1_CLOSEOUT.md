# Content Factory v1 Closeout

## Status

Content Factory v1 is closed around the stable daily ops baseline. The recommended daily command is:

```bash
make stable-daily-ops
```

## What v1 Covers

- Source, evidence, topic, and content production pipelines.
- Agent review, versioning, final candidate, and manual publishing preparation.
- Methodology-driven generation and visual strategy planning.
- Live sidecar pilots with safety gates.
- Image asset metadata, copy packs, and visual checklists.
- Content queue, publishing calendar, trial readiness, and stable ops review.
- Operator acceptance checklist and workbench baseline.

## What v1 Does Not Do

- It does not publish.
- It does not call WeChat APIs.
- It does not enter a WeChat draft box.
- It does not scrape backend metrics.
- It does not generate images.
- It does not change prompt/config/rules automatically.

## Next Phase

Phase 26 should be a real operator acceptance trial with manual feedback, content quality blocking issue repair, and manual publishing dry run with copy pack.
